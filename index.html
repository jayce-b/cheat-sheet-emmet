My Cheat Sheet : Emmet

Bienvenue sur ma page cheat sheet sur Emmet. Vous pourrez y retrouver beaucoup de raccourcis utiles.


Qu'est-ce qu'Emmet ?

Il s'agit d'un plugin pour éditeur de texte (Perso je l'utilise avec Sublime Text). Il est très utile pour augmenter la productivité et il évite la saisie d'un code rébarbatif, notamment les listes, les tableaux et même le CSS. En bref, vive Emmet !

Tout d'abord il est important de savoir qu'une fois les raccourcis Emmet tapés, il faut les valider soit par la touche TAB, soit par CTRL+E (selon les préférences utilisateur).

Emmet et le HTML

utiliser > pour ajouter un élément enfant (child)

utiliser + pour ajouter un élément frère (sibling)

utiliser * pour multiplier les éléments

utiliser . pour ajouter un nom de classe

utiliser () pour grouper des éléments multiples

utiliser {} pour insérer du texte dans un élément

utiliser $ pour ajouter une numérotation

utiliser $@X commencer la numérotation à X

utiliser $@- pour mettre la numérotation dans l'ordre décroissant

utiliser ^ pour remonter d'un élément enfant

ajouter un attribut avec [attr_name="valeur"]
